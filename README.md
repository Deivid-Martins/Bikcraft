# Bikcraft
# English
Site created with a focus on improving skills involving responsiveness in web development, site made with several semantic tags to facilitate use for both people without disabilities and those who have them.
# Português(Brasil)
Site criado com foco em aprimorar habilidades envolvendo responsividade em desenvolvimento web, site feito com diversas tags semânticas para facilitar tanto o uso para pessoas sem deficiência quanto para àquelas que possuem.
